# Documentacion / Base

Infraestructura profesional para NexTask — Plataforma IT Service Desk.

## Contenido

Este directorio documenta la base infraestructural del proyecto:

- **Decisiones tecnológicas:** por qué Next.js, React, TypeScript, Tailwind, PostgreSQL, Docker, pnpm
- **Estructura modular:** cómo está organizado `src/`, `infra/`, `docs/`
- **Configuración profesional:** ESLint, Prettier, TypeScript strict, Tailwind corporativo
- **Tooling:** husky, lint-staged, health checks, Docker Compose
- **Scripts y automatización:** desarrollo, build, validación, Docker

- **Fase actual:** Infraestructura + tooling limpio y profesional

- **Base sólida:** Estructura escalable, configuración moderna, calidad asegurada
